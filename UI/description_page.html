Welcome to Report2Plot

An automated academic analytics platform. Converted report files are cached locally, student score records are extracted using Gemini, and styled PowerPoint slide decks and Matplotlib charts are rendered instantly.

⚙️

How it works

1. Place Files: Put your PDF/image report cards in the input/ directory.

2. Scan Input: Click Analyze Files in Step 1 to auto-detect subjects and class list registries.

3. Configure & Run: Select student name, choose fields, choose output format, and run the pipeline!
